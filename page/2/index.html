  • rem单位详解及应用

    Mar 27, 2015

    rem（font size of the root element）是指相对于根元素的字体大小的单位。
    我们经常会看到rem值设置为62.5%，

    html {font-size: 62.5%;}

    这是因为现在浏览器中的默认的字体大小都是16px，62.5%*16px=10px;所以1rem=10px;方便计算。但是这有一个bug,就是中文版的chrome支持的最小字体是12px,那么1rem最小值就是12px,而不是10px了。
